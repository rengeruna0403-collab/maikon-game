# 舞昆アパート経営 - GAS 経営ダッシュボード

## セットアップ手順

### 1. GAS エディタに貼り付け

1. Google スプレッドシートを開く
2. **拡張機能 > Apps Script** を開く
3. `Code.gs` の中身をすべて貼り付けて **保存（Ctrl+S）**

### 2. 初回セットアップ実行

1. 関数リストから `setupAll` を選択して **▶ 実行**
2. 権限許可ダイアログが出たら「許可」を押す
3. 全シートが自動生成される

### 3. ウェブアプリとしてデプロイ

1. **デプロイ > 新しいデプロイ**
2. 種類: **ウェブアプリ**
3. アクセスできるユーザー: **全員**
4. 発行されたURLをコピー → `index.html` の GAS_URL 定数を更新

### 4. 自動更新トリガー設定（任意）

1. 関数 `setTimeTrigger` を実行
2. 30分ごとに Dashboard・CRM・Analytics が自動更新される

## シート構成

| シート | 用途 | 対象 |
|--------|------|------|
| **Dashboard** | KPIカード（スクリプト更新） | 社長 |
| **CRM** | 見込み客一覧・優先度付き | 営業担当 |
| **Analytics** | グラフ用集計データ | 開発者 |
| **Leads** | リード生データ（条件付き書式） | 全員 |
| **choice** | イベント選択ログ | 開発者 |
| **monthly** | 月次サマリーログ | 開発者 |
| **debug** | テストデータ確認 | 開発者 |

| 色 | 条件 |
|----|------|
| 🔴 薄赤 | 大家＋舞昆 両方興味あり（Hot Lead） |
| 🟠 薄オレンジ | 大家に興味あり |

## Dashboard KPI一覧

**ゲーム状況（社長用）**
- 総プレイ人数 / 今日のプレイ人数
- 1年目クリア人数 / クリア率
- 平均プレイ日数 / 平均店舗数

**ゲーム平均値**
- 平均ブランド力 / 平均地域評価
- 平均銀行信用度 / 平均所持金
- 平均入居率 / 平均物件数

**舞昆（マーケティング）**
- 舞昆興味あり人数
- おすすめ商品ランキング TOP3

**大家（営業）**
- 大家興味あり人数
- 人気オーナータイプ TOP2

**CRM**
- LINE登録数 / メール登録数
- Hot Lead件数 / 未対応件数
- 今日の新規Lead数

## カスタムメニュー

スプレッドシートを開くと上部に **🍙 舞昆ダッシュボード** メニューが追加される。

- `▶ 初回セットアップ` — 全シート・ヘッダー初期化
- `🔄 Dashboard/CRM/Analytics更新` — 手動更新
- `📊 グラフ再生成` — Analyticsシートのグラフを再作成
- `⏰ 自動更新トリガー設定` — 30分ごとの自動更新を有効化
